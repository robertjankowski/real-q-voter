src.real_q_voter.simulation.ba_graph_with_random_opinion(n, m=8) → networkx.classes.graph.Graph¶

Construct BA network with randomly generated opinions

src.real_q_voter.simulation.run(g: networkx.classes.graph.Graph, q: int, p: float, eta=0.2, n_iteration=1000, preference_sampling=False, majority_conform=False, save_plot_graphs=False)¶

Perform simulation of q-voter model on g graph

Parameters

  • g – nx.Graph (directed or undirected)

  • q – Number of agents in q-voter model

  • p – Independence factor

  • eta – Probability of conformity flip

  • n_iteration – Number of simulation steps

  • preference_sampling – Take into account degree of neighbours nodes while sampling for q nodes

  • majority_conform – True: In conformity state use majority votes of neighbors. False: All agents should have the same opinions.

  • save_plot_graphs – Save plot of network (10 snapshots)?

Returns

(mean_opinions: list, weighted_mean_opinions: list)

src.real_q_voter.simulation.run_simulation(g: networkx.classes.graph.Graph, q: int, p_range: list, n_iteration: int, save_plot_graphs=False)¶

Run multiple q-voter simulation for given p_range values

Parameters

  • g – nx.Graph

  • q – Number of agent in q-voter model

  • p_range – Range of p (independence factor)

  • n_iteration – Number of iterations of the single run (per one p value)

  • save_plot_graphs – Save images of network?

Returns

Tuple of mean opinions, each list has list per p value, e.g., mean_opinion = [[values_for_p=0.1], [values_for_p=0.2], …, ]
